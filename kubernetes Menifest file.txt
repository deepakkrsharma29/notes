SERVICES
1. NodePort
2. Cluster IP
3. Load Balancer

1. Node Port (vim svc.yml)
step 1 --> Firstly run a Deployment file with containing website host Ex.- image: rupeshsaini09/website
step 2 --> 
apiVersion: v1
kind: Service
metadata:
        name: mywebsite
        labels:
                app: website
spec:
        type: NodePort
        ports:
                - targetPort: 80
                  port: 80
                  nodePort: 30080
        selector:
                app: webapp

2. Cluster IP (vim csvc.yml)
step 1 --> Firstly run a Deployment file with containing website host Ex.- image: rupeshsaini09/website
step 2 -->
